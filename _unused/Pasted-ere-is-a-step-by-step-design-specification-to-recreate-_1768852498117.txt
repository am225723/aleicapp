ere is a step-by-step design specification to recreate the "Aleic" login screen exactly.

### 1. The Color Palette

* **Background Base:** Deep Charcoal/Midnight Blue (`#1A1A24` to `#242430`)
* **Gold Accent (Text/Lines):** Metallic Gold (`#D4AF37`) or a gradient from `#FDD663` to `#C5991A`.
* **Glow Color:** Amber/Orange glow (`#FFBF00` with 40% opacity).
* **Input Field Background:** Transparent white (`rgba(255, 255, 255, 0.05)`).
* **Text Color (Primary):** White or Off-White (`#F0F0F0`).
* **Text Color (Placeholder):** Muted Grey (`#888888`).

### 2. The Visual Layers

You must stack these elements using CSS `z-index` or Absolute Positioning.

**Layer 1: The Background Image**

* Find a photo of a couple/hands similar to the original reference.
* **Crucial Step:** Apply a dark overlay. In CSS, put a black `div` over the image with `opacity: 0.8` or `0.9`. This pushes the photo into the shadows so the gold pops.

**Layer 2: The "Line Art" Illustration**

* Place the gold outline illustration (the faces/heart) centered.
* **The Glow Effect:** Apply a drop-shadow filter to the image file itself.
* *CSS:* `filter: drop-shadow(0 0 10px rgba(212, 175, 55, 0.6));`



**Layer 3: The Frosted Glass Container**
This is the box holding the inputs.

* **Effect:** The "Frosted" look relies on background blur.
* *CSS:*
```css
background: rgba(255, 255, 255, 0.05); /* Very faint white */
backdrop-filter: blur(10px); /* This blurs whatever is behind the box */
-webkit-backdrop-filter: blur(10px); /* For Safari */
border: 1px solid rgba(255, 255, 255, 0.1); /* Subtle border */
border-radius: 20px;

```





### 3. Typography & Text Effects

**The Title ("Aleic")**

* **Font:** Use a script font like **"Great Vibes"**, **"Playball"**, or **"Allura"** (available on Google Fonts).
* **Gold Gradient Text:** To get that metallic look on the letters:
* *CSS:*
```css
background: linear-gradient(to bottom, #FDD663, #C5991A);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
filter: drop-shadow(0px 2px 4px rgba(0,0,0,0.5)); /* Shadow for depth */

```





**The Subtitle ("Assisted Learning...")**

* **Font:** A clean, geometric sans-serif like **Montserrat** or **Raleway**.
* **The Size Trick:** To make the letters A, L, E, I, C larger, you have to wrap them in `span` tags in your HTML.
* *HTML Concept:*
```html
<p class="subtitle">
   <span class="large-letter">A</span>SSISTED 
   <span class="large-letter">L</span>EARNING FOR 
   <span class="large-letter">E</span>MPATHETIC AND 
   <span class="large-letter">I</span>NSIGHTFUL 
   <span class="large-letter">C</span>OUPLES
</p>

```


* *CSS:*
```css
.subtitle { font-size: 12px; letter-spacing: 2px; color: #D4AF37; text-transform: uppercase; }
.large-letter { font-size: 16px; font-weight: bold; } /* 30% larger */

```





### 4. UI Elements (Inputs & Buttons)

**Input Fields**

* Remove default borders. Use only a bottom border (`border-bottom: 1px solid #555`).
* Background should be transparent.

**Sign In Button**

* **Style:** Pill-shaped (`border-radius: 50px`).
* **Border:** 1px solid Gold.
* **Text:** Gold.
* **Hover State:** Fill the button with Gold, change text to Dark Grey.

**Buttons at the bottom**

* Since you removed "Join as Therapist", center the "Join as Couple" button.
* Make it less prominent (ghost button style) with a thinner border and lower opacity text compared to the main Sign In button.

### Summary Checklist for Developers

If you are handing this off to a coder, give them this list:

1. **Font Family:** Script font for headers, Sans-Serif for body.
2. **Backdrop-filter:** Use `blur(10px)` for the login container.
3. **Specific Styling:** First letters of the subtitle acronym must be `1.3em` sized compared to the rest.
4. **Assets:** Provide the gold line-art SVG with a transparent background.
